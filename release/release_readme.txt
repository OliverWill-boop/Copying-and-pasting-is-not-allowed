Auto Input Helper

Files:
- AutoInputHelper.exe

Usage:
1. Double-click AutoInputHelper.exe
2. Paste or type the text
3. Click Start
4. Move the cursor to the target input before the countdown ends

Notes:
- This build includes the branded app icon and version metadata.
- The log file auto_input_gui.log will be created next to the EXE.
- Some security software may scan the EXE the first time it runs.
